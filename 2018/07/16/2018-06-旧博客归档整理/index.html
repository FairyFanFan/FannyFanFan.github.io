拓展运算符

const a = [1,2,3];

console.log(...a);  //1 2 3

function add(x,y,z){return x+y+z;};console.log(add(...a)); // 6

const brr = [0,0] ; 
brr.push(...a);console.log(brr); // [0, 0, 1, 2, 3]

const [first, ...rest] = [1, 2, 3, 4, 5];
first // 1
rest  // [2, 3, 4, 5] // 如果将扩展运算符用于数组赋值，只能放在参数的最后一位，否则会报错 const [...butLast, last] = [1, 2, 3, 4, 5];// 报错

在这个过程中还发现了一个问题，ref 和v-if问题，找不到$ref，渲染先后问题?
解决：ref 需要在dom渲染完成后才会有,在使用的时候确保dom已经渲染完成. 比如在 生命周期 mounted(){} 钩子中调用 或者在 this.$nextTick(()=>{}) 中调用
